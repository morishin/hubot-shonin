# hubot-shonin

A hubot script that gives you 承認

See [`src/shonin.coffee`](src/shonin.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-shonin --save`

Then add **hubot-shonin** to your `external-scripts.json`:

```json
["hubot-shonin"]
```

## Sample Interaction

```
you>> hubot 承認
hubot>> 最高!!!!フゥ〜〜〜〜〜!!!!
```
